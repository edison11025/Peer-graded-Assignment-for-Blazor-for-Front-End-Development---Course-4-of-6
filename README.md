# EventEaseApp2

A Blazor WebAssembly project built as part of the Coursera course **Blazor for Front-End Development (Course 4 of 6)**.  
EventEaseApp2 is designed to manage events, sessions, and user registrations with a clean, scalable architecture.

---

## 🚀 Features
- Event listing and details pages
- User registration form
- Session tracking and test page
- Responsive layout with Bootstrap
- Organized project structure (Components, Services, Models, Pages)

---

## 🛠️ Prerequisites
- [.NET 8 SDK](https://dotnet.microsoft.com/download) (or the version required by the course)
- Visual Studio 2022 / Visual Studio Code

---

## ⚙️ Setup Instructions
1. Clone the repository:  ```bash
   git clone https://github.com/edison11025/Peer-graded-Assignment-for-Blazor-for-Front-End-Development---Course-4-of-6.git
   cd Peer-graded-Assignment-for-Blazor-for-Front-End-Development---Course-4-of-6

2. Restore dependencies:
dotnet restore

3. Build the project:
dotnet build

4. Run the project:
dotnet run
